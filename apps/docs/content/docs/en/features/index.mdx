`@chiastack/features` provides a series of Zustand-based state management feature modules, including chat, todo, tic-tac-toe, and other features, along with practical utility functions.

## Peer Dependencies

`@chiastack/features` requires the following peer dependencies:

- `react` >= 18.3.0
- `react-dom` >= 18.3.0
- `zustand` >= 5.0.0
- `zod` >= 3.24.0
- `@chiastack/utils` >= 2.0.2

## Basic Concepts

All feature modules are built on top of Zustand state management library and provide the following features:

- **TypeScript Support**: Complete type definitions
- **React Context Integration**: Use via Provider and Hook
- **DevTools Support**: Built-in Redux DevTools integration
- **Modular Design**: Each feature is an independent module

## Usage Example

### Todo Store

```typescript
import { TodoStoreProvider, useTodoStore } from '@chiastack/features/todo/store';

function App() {
  return (
    <TodoStoreProvider>
      <TodoList />
    </TodoStoreProvider>
  );
}

function TodoList() {
  const todos = useTodoStore((state) => state.items);
  const addTodo = useTodoStore((state) => state.addTodo);
  
  return (
    <div>
      {todos.map((todo) => (
        <div key={todo.id}>{todo.text}</div>
      ))}
      <button onClick={() => addTodo('New Task')}>Add</button>
    </div>
  );
}
```
