`@chiastack/ui` is a React UI component library that provides reusable UI components, with a focus on trading charts (TradingView-style) and error handling functionality.

## Installation

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab value="npm">
    ```bash
    npm install @chiastack/ui
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add @chiastack/ui
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @chiastack/ui
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add @chiastack/ui
    ```
  </Tab>
</Tabs>

## Basic Usage

```tsx
import { Chart } from '@chiastack/ui/trading-chart/chart';
import { ErrorBoundary } from '@chiastack/ui/error-boundary';

function App() {
  return (
    <ErrorBoundary fallback={<div>Something went wrong</div>}>
      <Chart>
        {/* Chart components */}
      </Chart>
    </ErrorBoundary>
  );
}
```
