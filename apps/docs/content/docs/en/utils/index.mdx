`@chiastack/utils` provides a series of practical utility functions covering date handling, error handling, type checking, delayed execution, and more.

## Installation

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab value="npm">
    ```bash
    npm install @chiastack/utils
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add @chiastack/utils
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @chiastack/utils
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add @chiastack/utils
    ```
  </Tab>
</Tabs>
