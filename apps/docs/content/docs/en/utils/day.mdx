title: Day

The `day` module provides date and time handling functionality based on dayjs, including multiple common plugins and localization support.

## Installation

```bash
npm install @chiastack/utils
```

## Import

```typescript
import { dayjs, initDayjs } from '@chiastack/utils/day';
```

## API

### `initDayjs`

Initialize dayjs language and timezone settings.

```typescript
initDayjs(locale?: string, timezone?: string): void
```

**Parameters:**
- `locale` (optional): Language code, defaults to `"en"`
- `timezone` (optional): Timezone, defaults to `"UTC"`

**Example:**

```typescript
import { initDayjs } from '@chiastack/utils/day';

// Set Traditional Chinese and Taipei timezone
initDayjs('zh-tw', 'Asia/Taipei');
```

### `dayjs`

dayjs instance with the following plugins loaded:
- `localizedFormat` - Localized formatting
- `utc` - UTC time handling
- `tz` - Timezone handling
- `duration` - Duration handling
- `relativeTime` - Relative time
- `weekOfYear` - Week number calculation
- `advancedFormat` - Advanced formatting

**Example:**

```typescript
import { dayjs } from '@chiastack/utils/day';

// Format date
dayjs().format('YYYY-MM-DD HH:mm:ss');

// Use timezone
dayjs.tz('2024-01-01', 'Asia/Taipei');

// Relative time
dayjs().fromNow(); // "2 hours ago"

// Duration
dayjs.duration(2, 'hours').asMinutes(); // 120
```

## Supported Languages

- `en` - English (default)
- `zh-tw` - Traditional Chinese

## Dependencies

This module requires `dayjs` as a peer dependency:

```bash
npm install dayjs
```
