The `is` module provides a series of type guard functions for runtime type checking of values.

## Installation

```bash
npm install @chiastack/utils
```

## Import

```typescript
import {
  isNumber,
  isPositiveNumber,
  isNegativeNumber,
  isString,
  isURL,
  isUrl,
  isURLInstance,
  isAbortError,
} from '@chiastack/utils/is';
```

## API

### `isNumber`

Check if a value is a number.

```typescript
isNumber(value: unknown): value is number
```

**Example:**

```typescript
import { isNumber } from '@chiastack/utils/is';

if (isNumber(value)) {
  // TypeScript knows value is of type number
  console.log(value + 1);
}
```

### `isPositiveNumber`

### `isURL`

Check if a value is a valid URL string or URL instance.

```typescript
isURL(value: unknown): value is URL
```

**Example:**

```typescript
import { isURL } from '@chiastack/utils/is';

if (isURL(value)) {
  // value is a URL instance
  console.log(value.hostname);
}
```

### `isUrl`

Check if a value is a valid URL, optionally checking the protocol.

```typescript
isUrl<T, TStrict extends boolean = false>(
  url: T,
  options?: IsUrlOptions<TStrict>
): boolean
```

**Parameters:**
- `url`: Value to check
- `options` (optional):
  - `allowedProtocols`: Array of allowed protocols, defaults to `["http", "https"]`
  - `strict`: Whether to enforce checking as URL instance

**Example:**

```typescript
import { isUrl } from '@chiastack/utils/is';

// Basic check
isUrl('https://example.com'); // true
isUrl('invalid-url'); // false

// Check protocol
isUrl('ftp://example.com', {
  allowedProtocols: ['http', 'https']
}); // false

isUrl('https://example.com', {
  allowedProtocols: ['http', 'https']
}); // true

// Strict mode (must be URL instance)
const url = new URL('https://example.com');
isUrl(url, { strict: true }); // true
isUrl('https://example.com', { strict: true }); // false
```

### `isURLInstance`

Check if a value is a URL instance.

```typescript
isURLInstance(url: unknown): url is URL
```

**Example:**

```typescript
import { isURLInstance } from '@chiastack/utils/is';

const url = new URL('https://example.com');
if (isURLInstance(url)) {
  console.log(url.hostname);
}
```

### `isAbortError`

Check if an error is an abort error (AbortError or TimeoutError).

```typescript
isAbortError(error: unknown): error is DOMException
```

**Example:**

```typescript
import { isAbortError } from '@chiastack/utils/is';

try {
  await fetch('/api/data', { signal: abortController.signal });
} catch (error) {
  if (isAbortError(error)) {
    console.log('Request aborted');
  } else {
    console.error('Other error:', error);
  }
}
```
