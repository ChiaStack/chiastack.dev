`day` 模組提供了基於 dayjs 的日期和時間處理功能，包含多個常用插件和本地化支援。

## 安裝

```bash
npm install @chiastack/utils
```

## 導入

```typescript
import { dayjs, initDayjs } from '@chiastack/utils/day';
```

## API

### `initDayjs`

初始化 dayjs 的語言和時區設定。

```typescript
initDayjs(locale?: string, timezone?: string): void
```

**參數：**
- `locale` (可選): 語言代碼，預設為 `"en"`
- `timezone` (可選): 時區，預設為 `"UTC"`

**範例：**

```typescript
import { initDayjs } from '@chiastack/utils/day';

// 設定繁體中文和台北時區
initDayjs('zh-tw', 'Asia/Taipei');
```

### `dayjs`

dayjs 實例，已載入以下插件：
- `localizedFormat` - 本地化格式
- `utc` - UTC 時間處理
- `tz` - 時區處理
- `duration` - 時間長度處理
- `relativeTime` - 相對時間
- `weekOfYear` - 週數計算
- `advancedFormat` - 進階格式

**範例：**

```typescript
import { dayjs } from '@chiastack/utils/day';

// 格式化日期
dayjs().format('YYYY-MM-DD HH:mm:ss');

// 使用時區
dayjs.tz('2024-01-01', 'Asia/Taipei');

// 相對時間
dayjs().fromNow(); // "2 hours ago"

// 時間長度
dayjs.duration(2, 'hours').asMinutes(); // 120
```

## 支援的語言

- `en` - 英文（預設）
- `zh-tw` - 繁體中文

## 依賴

此模組需要 `dayjs` 作為 peer dependency：

```bash
npm install dayjs
```
