`@chiastack/utils` 提供了一系列實用的工具函式，涵蓋日期處理、錯誤處理、型別檢查、延遲執行等功能。

## 安裝

```bash
npm install @chiastack/utils
```

## 基本用法

```typescript
import { delay, tryCatch, isNumber } from '@chiastack/utils';

// 延遲執行
await delay(1000);

// 安全的錯誤處理
const result = await tryCatch(fetch('/api/data'));

// 型別檢查
if (isNumber(value)) {
  console.log(value + 1);
}
```
