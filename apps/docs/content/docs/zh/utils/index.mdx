`@chiastack/utils` provides a series of practical utility functions covering date handling, error handling, type checking, delay execution, and more.

## Installation

```bash
npm install @chiastack/utils
```

## Basic Usage

```typescript
import { delay, tryCatch, isNumber } from '@chiastack/utils';

// Delay execution
await delay(1000);

// Safe error handling
const result = await tryCatch(fetch('/api/data'));

// Type checking
if (isNumber(value)) {
  console.log(value + 1);
}
```
