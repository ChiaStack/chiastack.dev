本指南將幫助您快速開始使用 Chia Stack 套件。

## 安裝

Chia Stack 中的每個套件都可以獨立安裝和使用。

### Utils

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab value="npm">
    ```bash
    npm install @chiastack/utils
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add @chiastack/utils
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @chiastack/utils
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add @chiastack/utils
    ```
  </Tab>
</Tabs>
